NanoGL

Global NanoGL module. For convenience, all WebGL enums are stored as properties of NanoGL (e.g. NanoGL.FLOAT, NanoGL.ONE_MINUS_SRC_ALPHA).

(static) createApp(canvas, contextAttributesopt)

Create a NanoGL app. The app is the primary entry point to NanoGL. It stores the canvas, the WebGL context and all WebGL state.
