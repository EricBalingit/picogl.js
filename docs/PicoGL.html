Namespace: PicoGL

PicoGL

Global PicoGL module. For convenience, all WebGL enums are stored as properties of PicoGL (e.g. PicoGL.FLOAT, PicoGL.ONE_MINUS_SRC_ALPHA).
Properties:
Name Type Description
version string Current PicoGL version.
TEXTURE_INTERNAL_FORMAT object Map of framebuffer texture formats to internal formats.
TYPE_SIZE object Map of data types to sizes in bytes.
WEBGL_INFO object WebGL context information.
TEXTURE_UNIT_MAP object Map of texture unit indices to GL enums, e.g. 0 => gl.TEXTURE0.

(static) createApp(canvas, contextAttributesopt)

Create a PicoGL app. The app is the primary entry point to PicoGL. It stores the canvas, the WebGL context and all WebGL state.
Parameters:
Name Type Attributes Description
canvas DOMElement The canvas on which to create the WebGL context.
contextAttributes Object <optional>
 Context attributes to pass when calling getContext().
